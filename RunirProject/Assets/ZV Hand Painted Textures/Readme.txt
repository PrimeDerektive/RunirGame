~ZV Hand Painted Textures~



~ Contained within the archive are 25 tilable hand painted diffuse textures, along with 25 normal map textures, in 32 bit
     TGA format. All textures are 512x512.


~ Normal maps are tagged with "_NM" at the end of the file name.

  Example: "ZV_HP_WoodPlanks_05_NM"
